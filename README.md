# Create bootstrap node

```
$ ./l2 docker genesis bootstrap
```

# Create a new node

```
$ ./l2 docker genesis node
```

# start a node

```
$ ./l2 docker start
```

# create beaconscan
    
``` 
$ ./scan docker build
```

# start beaconscan

```
$ ./scan docker start
```

# stop beaconscan

```
$ ./scan docker stop
```
